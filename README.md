# WebApplications

- WebApp1 - An intro to Asp.Net Core Web Applications and its structure.
- MiddleWareAPP - Touches on Different Middlewares
- MinimalApiApp - Uses TypedResults and Results to return responses from a minimal API.
- MinimalApiApp2 - Uses Problem and ValidationProblem to return error responses from a minimal API.
- MinimalApiApp3 - Uses EndpointFilterFactory to endpoint filters which allow endpoint arguments not to be called in a specific order.
- MinimalApiApp4 - Uses IEndpointFilter for applying endpoint filters in a minimal API.
- MinimalApiApp5 - Uses MapGroups to group endpoints in a minimal API.
